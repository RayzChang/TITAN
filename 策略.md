# 🥇 策略 A：朋友的箱體突破（RangeBreakout v1.2）

> **核心概念：**
> 用「成交量錨定」找出歷史關鍵箱體，價格突破上緣 + MACD 確認 → 順勢做多；
> 跌破下緣 + 高量 → 箱體失效，重建新箱體。
>
> 這是個**慢頻率、大波段、高勝率**的趨勢突破策略。

## A.1 箱體建構演算法

### 初始化（兩種來源）

**方法 1：手動初始化（朋友手繪）**
```python
manual_boxes = {
    "BTC/USDT": {"floor": 60300, "ceilings": [74000, 76188]},
    "ETH/USDT": {"floor": 1740,  "ceilings": [2100, 2400]},
}
```
朋友肉眼看 K 線圖人工標出關鍵 floor 和 ceilings。

**方法 2：自動偵測（Volume-Anchored）**
```python
def _rebuild_box_from_data(self, symbol, df_1d):
    df = df_1d.iloc[:-1].tail(120).reset_index(drop=True)  # 取最近 120 根日線

    # 找成交量最大的 K 棒當錨點，但要確保後面至少還有 4 根日線
    vol = df['volume'].copy()
    anchor_idx = None
    for _ in range(len(df)):
        candidate = int(vol.idxmax())
        if candidate + 4 < len(df):
            anchor_idx = candidate
            break
        vol.iloc[candidate] = 0  # 排除這個候選，找次大量

    # 形成期 = 錨點 + 後續 5 根
    form_end = min(anchor_idx + 5, len(df) - 1)
    floor   = df['low'].iloc[anchor_idx:form_end + 1].min()
    ceiling = df['high'].iloc[anchor_idx:form_end + 1].max()

    return {'floor': floor, 'ceilings': [ceiling]}
```

**核心思想：** 高量 K 棒 = 強勢主力參與 → 該位置具有結構意義。

### 上緣延伸（假突破確認）

```python
def update_box(self, symbol, current_high, current_close, current_volume):
    box = self._boxes[symbol]
    top = box['ceilings'][-1]

    # 進入「上破追蹤」狀態
    if not box['tracking_breakout']:
        if current_close > top:  # 收盤站上最高 ceiling
            box['tracking_breakout']   = True
            box['breakout_max_high']   = current_high
            box['breakout_ref_ceiling'] = top

    # 已在追蹤中
    else:
        if current_high > box['breakout_max_high']:
            box['breakout_max_high'] = current_high
        if current_close < box['breakout_ref_ceiling']:
            # 收盤跌回原 ceiling 下方 → 確認假突破
            box['ceilings'].append(round(box['breakout_max_high'], 2))
            box['tracking_breakout'] = False
```

**含義：** 真實突破不算新層；只有「**突破又跌回**」才視為**假突破**並把該段最高點加成新一層 ceiling。

### 箱體失效偵測

```python
if current_close < floor:
    avg_vol_20d = df_1d.iloc[:-1]['volume'].tail(20).mean()
    if current_volume >= avg_vol_20d * 1.5:
        box['invalidated'] = True
        # 平倉後才會 _rebuild_box_from_data 重建
```

**含義：** 收盤跌破 floor + 量大於 20 日均量 1.5 倍 = 結構性崩壞。

## A.2 進場規則

### 多單 LONG（同一根已收 1H K 棒上同時成立）
```python
gold_cross = hist_now > 0 and hist_prev <= 0  # MACD 柱狀圖由負轉正
for ceil in ceilings:                         # 突破任一層 ceiling
    if close > ceil:
        return 'LONG'
```

### 空單 SHORT
```python
dead_cross = hist_now < 0 and hist_prev >= 0  # MACD 柱狀圖由正轉負
if close < floor and dead_cross:              # 跌破 floor + 死叉
    return 'SHORT'
```

### 防重複追單（R1）
進場後設 `needs_reset=True`，必須等：
- LONG：close 回到 base_ceiling 下方
- SHORT：close 回到 floor 上方

才允許下次進場。

## A.3 風險管理

### 止損 SL（Version B：固定不移動，永久結構支撐）
```python
LONG  : SL = floor × 0.99       # 永久固定
SHORT : SL = ceilings[-1] × 1.01  # 進場當下凍結（不隨後續延伸變動）
```

### 止盈（兩段式）
```
TP1：4H MACD 反向 → 平倉一半 + SL 移至開倉價（保本）
TP2：TP1 後 1H MACD 反向 → 全部平倉

特殊：TP1 觸發時 1H 也同時反向 → 直接全平，不留半倉
```

### 加倉（限一次，TP1 之前）
**做多加倉條件（同時滿足）：**
1. 連續 3 根已收 4H K 棒最低價 > 進場基準上緣
2. 最新已收日線 MACD 柱狀圖 > 0（多方持續狀態）

加倉後總部位 200 USDT 保證金，止損價不變只更新數量。

# 🥈 策略 B：MIA 的 V1 趨勢回調（TrendPullback）

> **核心概念：**
> 4H 看大方向，1H 等回調到 EMA20，RSI 從超賣翻揚 → 順勢進場。
>
> 這是個**高頻率、小波段、低勝率**的均值回歸策略。

## B.1 進場演算法（核心邏輯）

```python
def calculate_signals(self, df, symbol=''):
    # ── 4H 大方向 ──
    e20_4h = ema(self._df_4h['close'], 20)
    e50_4h = ema(self._df_4h['close'], 50)
    trend  = 'UP' if e20_4h.iloc[-2] > e50_4h.iloc[-2] else 'DOWN'

    # ── 1H 回調觸發 ──
    e20 = ema(df['close'], 20)
    rsi = rsi(df['close'], 14)
    last = df.iloc[-2]   # 已收完 K 棒
    ema_now = e20.iloc[-2]

    if trend == 'UP':
        touched   = last['low']  <= ema_now * 1.005   # 回調到 EMA20 附近
        recovered = last['close'] >  ema_now           # 收盤回站 EMA20 上方
        rsi_turn  = rsi_prev < 40 and rsi_now > rsi_prev  # RSI 從低位翻揚
        if touched and recovered and rsi_turn:
            return 'LONG'

    elif trend == 'DOWN':
        touched = last['high'] >= ema_now * 0.995    # 反彈到 EMA20 附近
        broken  = last['close'] <  ema_now            # 收盤跌回 EMA20 下方
        rsi_turn = rsi_prev > 60 and rsi_now < rsi_prev
        if touched and broken and rsi_turn:
            return 'SHORT'
    return 'HOLD'
```

**進場條件（4 個 AND）：**
1. **4H 大方向**：EMA20 vs EMA50（多/空判斷）
2. **1H 觸碰**：當根 low/high 觸到 EMA20 ± 0.5%
3. **1H 回站/跌破**：當根收盤穿越 EMA20
4. **RSI 反轉**：1H RSI 從超賣（< 40）翻揚 / 從超買（> 60）翻落

## B.2 風險管理

### 止損（動態，每筆不同）
```python
# 多單：SL = 進場價 × (1 - 距離)
# 距離 = |進場價 - 回調低點 × 0.995| / 進場價

# 空單：SL = 進場價 × (1 + 距離)
# 距離 = |反彈高點 × 1.005 - 進場價| / 進場價
```

通常範圍 **0.5% ~ 2%**（取決於回調深度）

### 止盈（固定 R-multiple）
```python
TP_distance = SL_distance × 2   # 1:2 風報比
# 多單：TP = 進場價 × (1 + 距離 × 2)
# 空單：TP = 進場價 × (1 - 距離 × 2)
```

# 🎯 給數據分析專家的核心問題

1. **兩個策略哪個有真實 edge？** 還是都靠運氣？

2. **這兩個策略應該 互補 還是 二選一？**
   - 直覺上：A（突破）和 B（回歸）方向相反 → 應該互補
   - 實測：兩者都在強趨勢市場（段 3、段 4）賺得多，盤整期都虧
   - 真互補嗎？還是只是換個包裝的同一個 beta？

3. **DOGE 在 V1 WF 段 1 -229% 怎麼解？**
   - 這個損益是「假設無限資金」算出來的
   - 實盤 100x 槓桿下，第一筆連虧就爆倉
   - 是否該丟掉 DOGE？還是改進策略？

4. **MIA 為什麼自動畫箱體跑不過朋友手繪？**
   - 是演算法不對嗎？
   - 還是「箱體」這個東西本質上需要主觀判斷？
   - 有沒有客觀指標能取代主觀眼力？

5. **MIA 該不該在 V1 之外設計新策略？**
   - 例如：均值回歸 in 盤整 + 趨勢跟隨 in 趨勢市
   - 例如：因子組合（動量 + 波動率 + 估值）
   - 還是該專注優化 V1？

6. **Top 20 幣全跑 vs 集中 3 幣（BTC、DOGE、ATOM）哪個對？**
   - BOSS 堅持「不剃除幣」
   - 但前 20 幣資金規模根本撐不起
   - Active/Shadow 制度（前 3 開倉、其他觀察）算合理折衷嗎？
